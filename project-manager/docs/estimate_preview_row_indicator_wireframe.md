# プレビュー行数インジケータ ワイヤー

見積プレビューの行数制御を、`現在 x / 22` と進捗バーで直感的に示すためのワイヤー。

## 1) 通常（0〜17行）

```
+--------------------------------------------------------------+
| プレビュー行数                                  現在 6 / 22 |
|                                                              |
| [███████░░░░░░░░░░░░░░░░] 27%                                |
|                                                              |
| ステータス: 通常                                              |
| このまま1ページに収まります                                  |
|                                                              |
| ※ 行数は「見出し1 + 明細 + 小計1」で計算                    |
+--------------------------------------------------------------+
```

- バー色: 緑系（例: `#16a34a`）
- 状態ラベル: `通常`

## 2) 注意（18〜22行）

```
+--------------------------------------------------------------+
| プレビュー行数                                 現在 20 / 22 |
|                                                              |
| [██████████████████████░░] 91%                               |
|                                                              |
| ステータス: 注意                                              |
| 上限に近づいています（残り2行）                              |
|                                                              |
| ※ 行数は「見出し1 + 明細 + 小計1」で計算                    |
+--------------------------------------------------------------+
```

- バー色: 黄/オレンジ系（例: `#f59e0b`）
- 状態ラベル: `注意`

## 3) 2ページ目作成（23行以上）

```
+--------------------------------------------------------------+
| プレビュー行数                                 現在 23 / 22 |
|                                                              |
| [████████████████████████] 100%                              |
|                                                              |
| ステータス: 2ページ目が作られました                          |
| 明細が22行を超えたため、2ページ構成で表示しています          |
|                                                              |
| ※ 行数は「見出し1 + 明細 + 小計1」で計算                    |
+--------------------------------------------------------------+
```

- バー色: 青系（例: `#2563eb`）
- 状態ラベル: `2ページ目が作られました`

## 表示ルール（確定）

- 進捗率は `min(現在行数 / 22, 100%)`
- 状態しきい値
  - `0〜17`: 通常（緑）
  - `18〜22`: 注意（黄/オレンジ）
  - `23以上`: 2ページ目作成済み（青）
- `23以上` は「超過エラー」ではなく「表示結果の通知」として扱う

## HTMLへの反映マッピング案

- 反映候補: `project-manager/docs/estimate_preview_wireframe.html`
- 挿入位置: プレビュー上部（現行の合計バー付近、または明細テーブルの直前）
- 推奨DOM構成
  - `.preview-row-indicator`
  - `.preview-row-indicator__head`（ラベル + `現在 x / 22`）
  - `.preview-row-indicator__track`（バー背景）
  - `.preview-row-indicator__fill`（幅を `%` 指定）
  - `.preview-row-indicator__status`（通常/注意/2ページ目作成）
  - `.preview-row-indicator__note`（計算式注記）
- 推奨状態クラス
  - `.is-normal`
  - `.is-warning`
  - `.is-page-created`
- 表示文言
  - 通常: `このまま1ページに収まります`
  - 注意: `上限に近づいています（残りN行）`
  - 23以上: `明細が22行を超えたため、2ページ構成で表示しています`
